## 2. CLα 양력선 기울기 산출

### 2.1 기존 방식 vs VLM 기반 산출

본 해석에서는 돌풍 하중배수 산출(Pratt 공식, §23.341)에 사용되는 양력선 기울기 CLα를
기존의 수동 입력 방식에서 **VLM AIC 행렬 기반 자동 산출** 방식으로 개선하였다.

**산출 원리:**
```
AIC 시스템:  w_j/V = Σ_i D_ji · γ_i
단위 받음각 (α = 1 rad):  w/V = -1  (균일 downwash)
순환 분포:  γ = D⁻¹ · (-1)

ΔCp_j = 2·γ_j / chord_j
CL = (1/S_ref) · Σ_j (ΔCp_j · Area_j)
CLα = CL / α = CL   (α = 1 rad)
```

VLM 해석은 CAERO 패널 전체(주날개 + 수평미익 + 수직미익)를 포함하므로,
산출되는 CLα는 **항공기 전체 양력선 기울기**이다.
이는 Pratt 돌풍 공식에서 요구하는 "airplane lift curve slope"에 부합한다.

### 2.2 CLα 비교 분석

| 산출 방법 | CLα (/rad) | 비고 |
|-----------|-----------|------|
| 박판 익형 이론 (2π) | 6.283 | 2D, 비압축성 |
| VLM (AEROS S_ref = 13.96 m²) | 6.391 | S_ref ≠ S_wing 기준 |
| **VLM (S_wing = 17.0 m², M=0)** | **5.247** | 날개면적 기준 정규화 |
| **VLM (S_wing = 17.0 m², M=0.235)** | **5.301** | 순항 마하수 적용, **채택** |
| 기존 수동 입력 | 5.500 | 이전 해석 |
| Prandtl 양력선 이론 (Wing only) | 5.027 | AR=8 가정 |

**분석:**
- VLM 산출값(5.30 /rad)은 기존 수동 입력(5.50)보다 **3.6% 낮음**
- Prandtl 양력선 이론(5.03, 날개만)보다 약 0.27 높은 것은 수평미익의 양력 기여분
- Prandtl-Glauert 압축성 보정에 의해 M=0.235에서 약 1% 증가
- AEROS S_ref(13.96 m²)와 날개면적(17.0 m²) 차이에 의한 기준면적 정규화 중요
  → W/S와 동일한 면적으로 정규화하여 Pratt 공식과 일관성 확보

**Mach 수 영향:**

| Mach | CLα (/rad) | P-G Factor |
|------|-----------|------------|
| 0.000 | 5.247 | 1.000 |
| 0.100 | 5.257 | 1.005 |
| 0.235 | 5.301 | 1.029 |
| 0.300 | 5.336 | 1.048 |

### 2.3 기준면적 정규화

VLM CLα 산출 시 기준면적(S_ref) 선택이 핵심이다.
Pratt 공식에서 CLα와 W/S는 동일한 기준면적을 사용해야 일관성이 보장된다:

```
Δn = (ρ₀ × V × CLα × K_g × U_de) / (2 × W/S)
μ_g = 2(W/S) / (ρ₀ × c̄ × CLα × g)
```

- **CLα × S** = 항공기 전체 dimensional lift-curve slope → 기준면적에 무관한 상수
- W/S에 `wing_area = 17.0 m²`를 사용하므로, CLα도 동일 면적으로 정규화
- `compute_rigid_clalpha(bdf_model, mach, ref_area=17.0e6)` 호출로 자동 보장

## 7. CLα 변경에 따른 설계하중 영향 분석

### 7.1 Gust 하중 변화

CLα = 5.50 → 5.30 변경에 의한 설계하중 영향:

| 항목 | CLα=5.50 (이전) | CLα=5.30 (VLM) | 변화 |
|------|----------------|----------------|------|
| Gust_VC+ nz | +4.53 | +4.44 | -2.0% |
| Wing root max M | ~33.0 MN·mm | 32.3 MN·mm | ~-2.1% |
| Fuselage max M | ~5.86 MN·mm | 5.74 MN·mm | ~-2.1% |

- CLα 3.6% 감소에 의해 돌풍 하중배수는 약 2% 감소
- 설계 최대 날개 루트 굽힘모멘트 약 2% 감소
- 비돌풍 케이스(대칭 기동, 착륙, yaw 등)는 CLα 변경에 무관

### 7.2 안전성 평가

- CLα 감소 → 돌풍 하중 감소 → 설계하중이 비보수적 방향으로 변화
- 그러나 VLM 기반 CLα가 이론적으로 더 정확한 값
- 기존 수동입력 CLα=5.50은 VLM 대비 보수적이었음
- Gust_VC+ 케이스가 여전히 wing sizing을 지배하는 critical case임을 확인

## 9. 결론 및 향후 과제

### 9.1 결론

1. KC-100 항공기에 대해 FAR Part 23 기반 50개 인증 하중케이스를 해석하여
   모든 케이스에서 100% 수렴을 달성하였다.

2. VLM AIC 행렬 기반 CLα 자동 산출 기능을 구현하여, 기존 수동 입력(5.50 /rad)을
   VLM 해석 기반 값(5.30 /rad)으로 대체하였다.

3. CLα 변경에 의해 돌풍 설계하중이 약 2% 감소하였으나, Gust_VC+가 여전히
   날개 구조의 critical sizing case임은 변함없다.

4. 주요 critical 설계하중 조건:
   - **날개:** Gust_VC+ (nz=+4.44) — 전단력 및 굽힘 지배
   - **수직미익:** Yaw VD (RUD=±15.5°) — 전 하중성분 지배
   - **동체:** Level Landing 및 Gust_VC+ — 굽힘 지배

### 9.2 향후 과제

- [ ] 다중 중량/CG 조건 추가 (OEW, Forward CG, Aft CG)
- [ ] ROLL/YAW 조종면 모델 BDF 반영 (현재 Warning 발생)
- [ ] Gust CLα에 대한 마하수 의존성 정밀 검토 (VB, VC, VD 각각 다른 마하에서 산출)
- [ ] 동적 돌풍 해석 (SOL 146) 추가 검토
- [ ] 피로하중 스펙트럼 생성
